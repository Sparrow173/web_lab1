Проект интернет-магазина на чистом HTML, CSS и JavaScript.

## Функциональность
- Каталог товаров
- Добавление/удаление из корзины
- Изменение количества товаров
- Автопересчет суммы
- Сохранение корзины в localStorage
- Оформление заказа через форму (модальное окно)
- Сообщение "Заказ создан!"
